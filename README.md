# 🏥 Disease Prediction Model

## 🏢 Internship

This project is part of the **CodeAlpha Machine Learning Internship (Task 4)**.

## 📌 Objective

To predict whether a person has a disease based on medical attributes using machine learning

## 📊 Dataset

The dataset includes features such as:

* Glucose
* Blood Pressure
* BMI
* Age
* Outcome (0 = No Disease, 1 = Disease)

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## 🤖 Model Used

* Random Forest Classifier

## 🔄 Workflow

1. Data Loading
2. Data Preprocessing
3. Train-Test Split
4. Model Training (Random Forest)
5. Prediction
6. Evaluation

## 📈 Evaluation Metrics

* Confusion Matrix
* Precision
* Recall
* F1-Score

## 📊 Results

* The model achieved good accuracy
* Performance improved compared to basic models
* Recall was analyzed to reduce missed disease cases

## 🔍 Conclusion

This project demonstrates how machine learning can be applied in healthcare for early disease detection using Random Forest.
